# 智能识别食物热量小程序

## 项目简介

智能识别食物热量小程序是一款基于AI图像识别技术的健康管理应用，帮助用户通过拍照识别食物并获取准确的热量和营养信息，实现科学的饮食管理。

## 功能特性

### 🎯 核心功能
- **AI智能识别**：拍照识别1000+种食物，90%+准确率
- **营养分析**：自动计算热量、蛋白质、脂肪、碳水化合物
- **饮食记录**：多餐记录，支持早餐、午餐、晚餐、加餐
- **数据分析**：日/周/月营养趋势分析
- **目标管理**：个性化健康目标设置和跟踪
- **智能推荐**：基于用户习惯的个性化建议

### 📱 技术特色
- **微信小程序原生开发**：TypeScript + Less
- **AI图像识别**：支持多食物同时识别
- **实时数据同步**：云端数据备份和同步
- **响应式设计**：适配不同屏幕尺寸
- **离线功能**：部分功能支持离线使用

## 开发环境

### 环境要求
- Node.js 16+
- 微信开发者工具
- TypeScript 4.0+

### 安装依赖
```bash
npm install
```

### 开发调试
1. 使用微信开发者工具打开项目
2. 选择 `miniprogram` 目录作为项目根目录
3. 点击编译运行
